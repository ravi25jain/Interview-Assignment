Prerequisuit eclipse Luna ide with javaSe1.8 installed on the system;

1. Export the Files in the workspace
2. search the main file to run the program.


"CsvJsonFileHandeling" and "JsonFileOperation" projects uses the external jar file
Add Jar File to these project to run these projects.
To Add Jar..

1. Export the file in Eclipse
2. Right click on the project in package explorer -> Build Path-> Configure Build Path
3. Select Libraries tab 
4. Click on Add External JARs
5. Select "simple-json-1.1.1.jar" file saved in the folder
6. Click on ok.
